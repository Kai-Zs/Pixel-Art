## ✨ 核心功能

### 🖼️ 画布系统

- **尺寸预设** - 16×16、32×32、64×64、128×128、256×256
- **自定义尺寸** - 8-512 像素，支持独立宽高
- **透明背景** - 棋盘格显示
- **撤销/重做** - 快捷键支持

### 🎨 绘图工具

- **三种模式**
  - ✏️ **绘制** - 自由绘制像素
  - 🖱️ **选择** - 矩形、魔棒、套索
  - 🧹 **擦除** - 精确擦除像素
- **智能光标样式** - 根据模式自动切换
- **魔棒容差** - 可调节选择精度

### 🌈 颜色系统

- **10种预设色板** - 基础、复古、扁平、马卡龙、霓虹、大地、海洋、日落、赛博朋克、灰阶
- **HEX输入** - 支持 #RGB 和 #RRGGBB
- **自定义色板** - 取色器添加，右键删除，LocalStorage 持久化

### 📚 图层系统

- 创建、删除、重命名图层
- 拖拽排序、透明度调节
- 显示/隐藏/锁定

### 🔍 缩放与导航

- **缩放** - Ctrl + 滚轮、顶部按钮、双指缩放（移动端）
- **平移** - 右键拖拽、空格 + 左键拖拽、画布外拖拽支持
- **重置视图** - 一键缩放适应
- **状态栏** - 显示尺寸、坐标、缩放比例

### 🎯 用户体验

- **可移动悬浮工具栏** - 带拖动抓手
- **Toast 提示** - 操作反馈
- **确认对话框** - 危险操作二次确认
- **响应式设计** - 适配各种屏幕

## 📖 使用指南

### 基础操作

1. 选择画布尺寸
2. 选择颜色
3. 点击或拖动绘制
4. 切换绘制/选择/编辑/擦除模式

### 选择与编辑

- **矩形选择**：选择模式下拖动框选
- **魔棒选择**：点击选择相似颜色，用容差滑块调节精度
- **套索选择**：自由绘制选区
- **移动选区**：选区内拖动
- **复制粘贴**：Ctrl+C / Ctrl+V
- **删除选区**：Delete 键

### 图层操作

右侧图层面板：创建、排序、调节透明度、显示/隐藏、锁定
