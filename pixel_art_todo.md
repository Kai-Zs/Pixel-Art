## 🎯 技术栈

- **框架**: React 18.2+
- **语言**: TypeScript 5.0+
- **构建工具**: Vite 7.2+
- **UI图标**: Lucide React
- **状态管理**: React Hooks
- **样式**: CSS-in-JS

## 📝 开发说明

### 当前版本

- **版本号**: beta2

### 核心架构

- 单文件组件架构（App.tsx）
- 基于 Canvas 的像素渲染
- React Hooks 状态管理
- LocalStorage 数据持久化

### 已知问题

- 大画布（256×256以上）性能有待优化
- 移动端部分操作体验需要改进
